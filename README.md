# みんなの菜園会計自動化システム

コミュニティガーデン「みんなの菜園」の会計を自動化するGoogle Apps Script（GAS）スクリプトです。  
レシート写真を Google Drive に入れると、Gemini API が自動でスプレッドシートに記入してくれます。

## 概要

- **処理内容**：支出レシート画像から日付・金額・費目を自動抽出して、スプレッドシートに記録
- **使用技術**：Google Apps Script + Gemini 2.5 Flash API
- **実行頻度**：1時間ごと（時間ベーストリガー）
- **対応フォーマット**：JPG, PNG 等の画像ファイル

## システム構成

```
Google Drive
├── みんなの菜園（全員アクセス可）
│   └── みんなの菜園会計_公開用（IMPORTRANGE参照）
│
└── みんなの菜園運営チーム（運営メンバーのみ）
    └── 会計管理
        └── 2026年度
            ├── みんなの菜園会計_2026年度（管理用・GASここに作成）
            ├── 01_支出記入用画像（レシート写真→ここに入れる）
            ├── 02_支出保存用画像（正常処理済み写真）
            └── 03_取込エラー画像（読み取り失敗時）
```

## セットアップ手順

### 1. Google Driveフォルダの作成

上記の構成に従ってフォルダを作成してください。各フォルダの ID が後ほど必要です。

### 2. スプレッドシートの準備

**管理用スプレッドシート** `みんなの菜園会計_2026年度` に以下のシートを作成します：

| シート名 | 内容 |
|---------|------|
| サマリー | 年間集計 |
| 明細 | 収支記録（GAS自動入力 + 手入力） |
| 使い方 | 運用手順・トラブル対応 |
| 設定 | APIキー・フォルダID・費目リスト（非表示推奨） |

### 3. 設定シートへの入力

**設定シート** の以下の項目を入力してください：

| 項目 | 値 | 例 |
|-----|-----|-----|
| Gemini APIキー | Google AI Studio で取得したAPIキー | `AIza...` |
| 01_支出記入用画像フォルダID | Google Drive フォルダID | `1a2b3c4d5e...` |
| 02_支出保存用画像フォルダID | Google Drive フォルダID | `2f3g4h5i6j...` |
| 03_取込エラー画像フォルダID | Google Drive フォルダID | `3k4l5m6n7o...` |
| スプレッドシートID | 管理用スプレッドシートのID | `1AbC...` |

**費目リスト（F列から記載）：**
```
種苗代
土・資材
農具・備品
イベント費
消耗品
会費
イベント収入
補助金・助成金
その他
```

### 4. GASコードの配置

管理用スプレッドシートで以下を実行します：

1. メニュー → **拡張機能** → **Apps Script** を開く
2. デフォルトのコードを全削除
3. [Code.js](./gas/Code.js) の内容をコピペして貼り付け
4. **Ctrl+S** で保存

### 5. トリガー設定

Apps Script エディタで：

1. 左側メニュー → **トリガー**
2. **トリガーを作成** ボタンをクリック
3. 以下の設定を入力：
   - **実行する関数**：`runHourly`
   - **デプロイを選択**：新規デプロイ
   - **イベントのソースを選択**：時間ベース
   - **時間ベーストリガーのタイプ**：1時間ごと
4. **保存** をクリック

## 明細シート構成

| 列 | 項目 | 説明 |
|----|------|------|
| A | No. | 1から始まる連番（GAS自動採番） |
| B | 日付 | YYYY-MM-DD 形式（レシートから自動抽出） |
| C | 種別 | 固定値「支出」 |
| D | 費目 | プルダウン（設定シートのリストから選択） |
| E | 内容（用途） | レシートから自動抽出 |
| F | 金額（税込）| 整数のみ、通貨記号なし |
| G | 備考 | 処理済み画像の Google Drive URL |

## ファイル処理フロー

```
01_支出記入用画像 に写真を入れる
         ↓
1時間ごとに GAS が実行
         ↓
Gemini API で画像解析
         ↓
【成功時】
  ✓ 明細シートに記入
  ✓ ファイルを 02_支出保存用画像 に移動
  ✓ ファイル名を「No1_20260315_種苗代_1200円.jpg」に変更
         ↓
【失敗時】
  ✗ ファイルを 03_取込エラー画像 に移動
  ✗ ファイル名を「ERROR_元ファイル名.jpg」に変更
  ✗ エラーメール送信（運営メンバーに通知）
```

## トラブルシューティング

### エラーメールが届いた場合

1. **Google Drive URL を確認**
   - フォルダID が正しいか、フォルダにアクセス権があるか確認
   
2. **Gemini APIキー を確認**
   - [Google AI Studio](https://aistudio.google.com/app/apikey) で新しいキーを取得（無料）
   - 設定シートで新しいキーに置き換え

3. **レシート画像の品質**
   - 画像が不鮮明な場合、解析に失敗する可能性があります
   - 再撮影して 01_支出記入用画像 に入れ直してください

4. **明細シートの構成**
   - ヘッダー行（3行）が存在するか確認
   - 列数・列の型が正しいか確認

### Gemini APIのモデルエラーが出た場合

Geminiのモデル名が廃止されている可能性があります。

1. [Google AI Studio](https://aistudio.google.com/) で利用可能なモデルを確認
2. Code.js の先頭にある `MODEL_NAME` を最新のモデル名に更新
   - 例：`const MODEL_NAME = 'gemini-2.5-flash';` → `'gemini-3.0-flash'` など

## 新年度の切り替え（年1回・4月）

1. 「2026年度」フォルダをコピーして「2027年度」に名前変更
2. スプレッドシートのデータ行を削除して繰越金額を設定
3. 01〜03フォルダが空か確認
4. 設定シートのフォルダIDを新年度のものに更新
